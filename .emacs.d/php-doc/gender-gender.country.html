Gender\Gender::country

(PECL gender >= 0.8.0)

Gender\Gender::country — 国を表すテキストを取得する

説明

public string Gender\Gender::country ( integer $country )

Gender クラスの定数から、国を表す文字列を取得します。

パラメータ

country

Gender クラスの定数で表される国 ID。

返り値

国名を文字列で返します。失敗した場合は false を返します。
